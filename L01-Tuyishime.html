This lab introduces us to basic HTML tags

The DOCTYPE Tag <!DOCTYPE html>

The doctype declaration is a short for Document Type Declaration (DTD). It is an
instruction to the web browser about what version of the markup language the page is
written in. With HTML5, the doctype declaration is only needed to enable the standard
mode for documents written using the HTML syntax.

The Meta CHARSET tag <meta>

The meta tag is typically used to provide structured metadata such as a document's
keywords, description, author name, character encoding, and other metadata.

The TITLE Tag <title>

The title tag is used to defines a title in the browser toolbar, and it provides a
title for the page when it is added to favorites. Also, the title tag displays a title
for the page in search-engine results.
